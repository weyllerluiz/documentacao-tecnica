HTML

HTML é a linguagem de marcação padrão para criar páginas web.

Ele descreve a estrutura de uma página usando uma série de elementos e tags.

  • <div>: Contêiner genérico
  • <p>: Parágrafo
  • <a>: Âncora ou link
  • <h1>: Cabeçalho nível 1
  • <img>: Imagem
<div>...</div>
CSS

CSS é usado para controlar o estilo e o layout das páginas web.

Permite aplicar estilos como cores, fontes e espaçamento aos elementos HTML.

  • Propriedades de Cor
  • Propriedades de Fonte
  • Propriedades de Espaçamento
body { background-color: #f0f0f0; }
JavaScript

JavaScript é a linguagem de programação que permite a criação de conteúdo dinâmico na web.

Ele pode atualizar e alterar tanto HTML quanto CSS, além de manipular dados.

  • Variáveis
  • Funções
  • Eventos
console.log('Hello, world!');
